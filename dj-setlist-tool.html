Virtual DJ

Auto-saves a dated XML after every session

  • Open Finder → Go → Home
  • Navigate to Documents → VirtualDJ → History
  • Find the .xml file with last night's date, e.g. 2026-03-07.xml
  • Drag it into the drop zone above
  • Open File Explorer
  • Navigate to Documents → VirtualDJ → History
  • Find the .xml file with last night's date
  • Drag it into the drop zone above
💡 Inside the app: File menu → History → right-click a session → Export

Rekordbox

Export your playlist as XML from the sidebar

  • Find your set playlist in the left sidebar
  • Right-click → Export Playlist → choose XML
  • Drag the saved file into the drop zone above
  • Find your set playlist in the left sidebar
  • Right-click → Export Playlist → choose XML
  • Drag the saved file into the drop zone above
💡 Full library: File → Export Collection in xml format

Serato DJ

Every session auto-saves as a binary .session file

  • Open Finder
  • Navigate to Music → _Serato_ → History → Sessions
  • Find the .session file from last night
  • Drag it into the drop zone above
  • Open File Explorer
  • Navigate to My Music → _Serato_ → History → Sessions
  • Find the .session file from last night
  • Drag it into the drop zone above
💡 Also: History panel (bottom left) → right-click session → Export as .csv

DJay Pro

Export from the History panel

  • Click the History icon (clock, top-right)
  • Right-click last night's session → Export as CSV
  • File lands in Downloads — drag into drop zone above
  • Tap the History button
  • Swipe left on the session → tap Export
  • AirDrop to your Mac → drag into drop zone above
💡 AirDrop: After export, tap AirDrop → Your Mac. Appears in Downloads instantly.
